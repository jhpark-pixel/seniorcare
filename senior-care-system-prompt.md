# 시니어 주거시설 입주자 통합 관리 시스템 — Claude Code 프롬프트

아래 프롬프트를 Claude Code에 그대로 입력하세요.

---

시니어(고령자) 주거시설 입주자 통합 관리 시스템을 풀스택으로 개발해줘.

## 기술 스택

- Frontend: React + TypeScript + Tailwind CSS
- Backend: Node.js + Express + TypeScript
- DB: SQLite (개발용) / PostgreSQL (운영용)
- ORM: Prisma
- 차트: Recharts
- 실시간 통신: Socket.IO (IoT 낙상 감지 알림용)
- PDF 출력: @react-pdf/renderer 또는 pdfkit
- 한국어 UI, 반응형 디자인

## 프로젝트 구조

```
/senior-care-app
  /client          (React SPA)
  /server          (Express API)
  /prisma          (DB schema + seed)
  /iot-simulator   (낙상 감지 시뮬레이터)
```

## 핵심 기능

### 1. 입주자 관리 (CRUD)

**기본 정보**
- 이름, 생년월일, 나이(자동계산), 성별, 호실 번호, 입주일
- 비상연락처 (이름, 관계, 전화번호) 최소 1명, 최대 3명
- 프로필 사진 업로드
- 입주 상태: 입주중 / 외출중 / 입원중 / 퇴소

**건강 정보**
- 신체: 키(cm), 체중(kg), BMI(자동계산)
- 바이탈: 혈압(수축기/이완기), 혈당(공복/식후), 콜레스테롤(총/HDL/LDL), 심박수, 체온
- 기저질환: 당뇨, 고혈압, 심장질환, 관절염, 치매/인지장애, 골다공증, 파킨슨, 뇌졸중 이력, 호흡기질환, 신장질환 — 복수 선택 + 직접 입력
- 복용 약물: 약물명, 용량, 복용 시간, 처방의
- 알레르기: 음식, 약물, 기타
- 식이제한: 저염식, 저당식, 저지방식, 연하곤란식(다진식/미음), 채식, 유당불내증, 투석식 등 복수 선택
- 운동 능력 등급: Level 1(자유보행) / Level 2(보조기구) / Level 3(휠체어) / Level 4(거동불편·침상)
- 인지 능력 등급: 정상 / 경도인지장애 / 중등도치매 / 중증치매
- 낙상 이력: 날짜, 장소, 부상 정도, 비고

---

### 2. 일일 건강 데이터 기록

- 매일 기록 가능: 혈압, 혈당, 체중, 심박수, 체온, 수면시간, 수분섭취량, 식사량(상/중/하), 배변 여부, 기분상태(5단계)
- 기록 입력 UI: 간편 입력 폼 (자주 쓰는 값 프리셋)
- 이상치 자동 감지: 정상 범위 벗어나면 즉시 경고 표시
- 기록 이력 테이블 + 시계열 차트 (일별/주별/월별 전환)

---

### 3. 건강 분석표 (대시보드)

**개인 분석표**
- 건강 종합 점수: 100점 만점 (BMI 20점, 혈압 20점, 혈당 20점, 콜레스테롤 15점, 활동성 15점, 수면/수분 10점)
- 각 항목별 정상(초록) / 주의(노랑) / 위험(빨강) 3단계 시각화
- 지난 30일 추이 라인 차트 (체중, 혈압, 혈당)
- 위험 요소 자동 태그: "고혈압 주의", "저체중 위험", "낙상 고위험", "탈수 주의" 등
- 종합 소견 텍스트 자동 생성

**전체 입주자 대시보드**
- 요약 카드: 총 입주자 수, 위험군 수, 오늘 바이탈 미입력자, 오늘 낙상 알림 건수
- 입주자 목록 테이블: 이름, 호실, 건강점수, 위험태그, 상태 — 정렬/필터/검색 가능
- 위험도별 필터: 전체 / 위험군만 / 주의군만
- 최근 이벤트 피드: 낙상 감지, 바이탈 이상, 긴급 호출 등

---

### 4. IoT 낙상 감지 시스템

**시뮬레이터 (/iot-simulator)**
- Node.js 스크립트로 가상 센서 데이터 생성
- 3축 가속도계(accelerometer) + 자이로스코프 데이터 시뮬레이션
- 정상 활동(걷기, 앉기, 눕기) vs 낙상 이벤트 패턴 구분
- 랜덤 또는 수동 트리거로 낙상 이벤트 발생 가능
- WebSocket(Socket.IO)으로 서버에 실시간 전송

**서버 처리**
- 낙상 감지 이벤트 수신 → DB 기록
- Socket.IO로 프론트엔드에 실시간 알림 push
- 알림 데이터: 입주자명, 호실, 발생시각, 센서 수치, 심각도(경고/긴급)

**프론트엔드 알림**
- 실시간 알림 배너 + 알림음
- 알림 센터: 전체 알림 이력, 읽음/미읽음 처리
- 낙상 상세: 발생 시각, 위치(호실/복도/화장실 등), 대응 상태(미대응/대응중/완료)
- 대응 기록 입력: 대응자, 대응 내용, 결과(이상없음/경상/병원이송 등)

**센서 장치 관리 (기본 CRUD)**
- 장치 ID, 설치 위치(호실 번호), 배터리 상태, 마지막 통신 시각
- 장치 상태: 정상 / 배터리 부족 / 통신 끊김

---

### 5. AI 맞춤 가이드 자동 생성

입력된 건강 데이터 + 기저질환 + 운동능력 + 인지능력을 종합 분석하여 입주자별 맞춤 가이드를 규칙 기반 알고리즘으로 자동 생성한다 (외부 API 호출 없이 자체 로직으로 구현).

**(a) 식단 가이드**
- 질환별 식이요법 규칙 적용 (당뇨→저당, 고혈압→저염, 신장→저단백 등)
- 복수 질환 시 교차 규칙 적용 (당뇨+고혈압 → 저당+저염)
- 주간 식단표 예시: 월~일, 아침/점심/저녁/간식
- 일일 영양소 목표: 칼로리, 단백질, 탄수화물, 지방, 나트륨, 식이섬유, 수분
- 권장 식품 / 주의 식품 / 금지 식품 목록
- 연하곤란 등급 반영 (일반식 → 다진식 → 미음)

**(b) 운동 가이드**
- 운동 능력 등급별 프로그램:
  - Level 1: 걷기 30분, 스트레칭, 밴드 운동, 균형 운동
  - Level 2: 보조기구 활용 보행 훈련, 앉아서 하는 상체 운동
  - Level 3: 휠체어 상체 운동, 관절 가동범위 운동
  - Level 4: 침상 스트레칭, 수동 관절 운동, 욕창 예방 체위 변환
- 기저질환 반영: 관절염→수중운동 권장, 심장질환→고강도 금지, 골다공증→충격운동 금지
- 낙상 위험도 반영: 고위험군은 균형 훈련 강화, 단독 운동 제한
- 일일 스케줄: 시간대, 운동 종류, 시간, 강도, 주의사항

**(c) 생활 습관 가이드**
- 수면: 권장 수면 시간, 수면 위생 팁 (질환별 맞춤)
- 수분 섭취: 일일 목표량 (신장질환 시 제한량 반영)
- 인지 활동: 인지 등급별 추천 (퍼즐, 독서, 회상치료, 음악치료 등)
- 사회 활동: 고립 방지를 위한 프로그램 참여 권장
- 정기 검진 스케줄: 질환별 필수 검진 항목 + 주기 + 다음 검진 예정일 리마인더

**(d) PDF 리포트 내보내기**
- 개인별 건강 분석표 + 맞춤 가이드 전체를 PDF로 출력
- 헤더: 시설명, 출력일, 입주자 정보
- 가족 공유용 / 의료진 참고용 2가지 템플릿

### 6. 시설 서비스 프로그램 추천 시스템

시설 내에서 운영하는 다양한 프로그램/서비스를 등록하고, 입주자의 건강 상태·능력·관심사에 맞춰 자동 추천한다.

**프로그램 관리 (관리자 CRUD)**
- 프로그램 정보: 이름, 카테고리, 설명, 담당자, 요일/시간, 장소, 정원, 현재 참여 인원
- 카테고리:
  - 건강/재활: 물리치료, 작업치료, 재활운동, 수중치료
  - 운동/체력: 요가, 태극권, 체조, 산책모임, 게이트볼
  - 인지/교육: 인지훈련, 컴퓨터교실, 독서모임, 외국어교실
  - 문화/여가: 미술치료, 음악치료, 영화감상, 원예치료, 서예, 노래교실
  - 사회/심리: 그룹상담, 회상치료, 종교활동, 봉사활동
  - 외부서비스: 이미용, 발관리, 건강강좌, 외부공연
- 참여 조건: 운동 능력 최소 등급, 인지 능력 최소 등급, 제외 질환
- 프로그램 상태: 모집중 / 진행중 / 종료 / 일시중단

**자동 추천 알고리즘**
- 입주자의 건강 데이터를 기반으로 적합한 프로그램 자동 매칭:
  - 운동 능력 등급 ≥ 프로그램 요구 등급
  - 인지 능력 등급 ≥ 프로그램 요구 등급
  - 기저질환과 제외 질환 충돌 없음
  - 건강 분석 결과 반영 (예: 낙상 고위험 → 균형 운동 프로그램 우선 추천, 우울 경향 → 사회/문화 프로그램 추천, 인지 저하 → 인지훈련 프로그램 추천)
- 추천 점수 산출: 적합도(40%) + 건강 필요도(40%) + 정원 여유(20%)
- 추천 결과 UI: 카드형 목록, 추천 이유 표시, "참여 신청" 버튼

**입주자별 프로그램 참여 관리**
- 현재 참여 중인 프로그램 목록
- 주간 시간표 캘린더 뷰
- 참여 이력 기록
- 출석 체크

**프로그램 효과 추적 (선택적)**
- 프로그램 참여 전후 건강 지표 비교 차트
- "이 프로그램 참여 후 균형감각 점수 15% 향상" 같은 인사이트

## 시드 데이터

데모용 시드 데이터를 반드시 포함해줘:

- 관리자 3명 (시설장 1, 간호사 1, 생활지도사 1)
- 입주자 5명 (다양한 건강 상태: 건강한 어르신 1명, 당뇨+고혈압 1명, 치매+낙상이력 1명, 관절염+골다공증 1명, 심장질환+신장질환 1명)
- 각 입주자별 최근 30일 바이탈 기록 (랜덤 생성, 질환 특성 반영)
- 시설 프로그램 10개 (카테고리별 2개씩)
- 낙상 이벤트 3건 (대응 기록 포함)
- IoT 장치 5개

## 개발 순서

1. Prisma 스키마 작성 + 마이그레이션
2. 시드 데이터 생성
3. Express API 엔드포인트 구현
4. React 페이지 구조 + 라우팅
5. 입주자 관리 (목록/등록/상세)
6. 건강 데이터 기록 + 차트
7. 건강 분석표 대시보드
8. IoT 낙상 감지 (시뮬레이터 + 실시간 알림)
9. 맞춤 가이드 생성 엔진
10. 시설 프로그램 관리 + 추천 시스템
11. PDF 리포트 내보내기
12. 관리자 인증 + 권한

DB 스키마부터 시작해서 위 순서대로 하나씩 구현해줘. 각 단계마다 동작 확인 가능하게 만들어줘.
